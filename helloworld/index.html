Print Hello World in different languages

  • Let’s learn how to print hello world in different programming languages

Shell Script

echo "Hello World!"

C

#include<stdio.h>

int main() {
  printf("Hello World!\n");
  return 0;
}

C++

#include<iostream>

int main() {
  std::cout << "Hello World!\n";
  return 0;
}

Java

class HelloWorld {
  public static void main(String[] args) {
      System.out.println("Hello, World!");
  }
}

Ruby

puts "Hello World!"

Python

print("Hello World!")

Groovy

println("Hello World!")

Go

package main

func main(){
  println("Hello World!")
}
